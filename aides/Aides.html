Bienvenue dans l'aide !

Généralité du site

Le contenu modifiable du site est situé dans le dossier content/ de ton arborescence.
Tu y retrouves donc :

  • Agenda
  • Photos
  • Prestations
  • Références

Pour ajouter ou enlever du contenu il faut que tu crées ou supprimes le fichier correspondant.

Exemple: le fichier content/prestation/JONGLAGE.md correspond à la prestation Jonglage.

Pour afficher du contenu il faut donc remplir ces fichiers. Le détail de ces paramètres est accessible dans les différentes rubriques du menu.

Les images sont situées dans le dossier static/img/ de ton arborescence.
Tu y retrouve donc :

  • Home
  • Prestations
  • References

En effet, ce sont les parties de ton site qui contiennent des images.

Généralité pour la publication

Le site est stocké à différents endroits :

  • En local sur ton ordinateur
  • Sur GitHub pour sauvegarder le contenu (backup) et mettre à jour le site (corriger des bugs, changer le design)
  • Chez l'hebergeur (ovh, goDaddy) qui est chargé de diffuser le site sur internet (manoel-mathis.fr par exemple)

Voici une liste (non exhaustive) de processus possible :

Ajouter du contenu

  1. Tu modifies le site sur ton ordinateur (ajout d'un fichier prestation par exemple)
  2. Tu peux visualiser les modifications en local (optionnel)
  3. Tu executes un script (cad double cliquer sur un fichier !) pour publier le site sur GitHub et l'hebergeur

Correction d'un bug

  1. Tu executes un script pour télécharger la nouvelle version du site depuis GitHub
  2. Tu executes un autre script pour publier le site sur gitHub et l'hebergeur

Le détail de ces commandes est disponible dans la rubrique publication du menu.
